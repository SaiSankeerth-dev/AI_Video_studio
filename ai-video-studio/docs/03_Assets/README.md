# Assets — drop files here

Hybrid mode: put anything you already have into the right subfolder; source the rest from the
links in `02_FIFA_2026_Edit/ASSET_SOURCES.md`.

- **Video_Clips/** — name files by slot, e.g. `01_stadium_aerial.mp4`, `08_city_montage_NYC.mp4`.
- **Audio_Music/** — your chosen track + SFX.
- **Voiceover/** — your recorded VO (or skip and use CapCut text-to-speech).
- **Images_Graphics/** — end-card stills, logos you own.

I did **not** auto-download files: the free stock sites require you to accept their licence at
download, and I won't pull anything copyrighted (FIFA footage/logos). The source list gives you
exact searches so each download is a few clicks. Log each file in `ASSET_LICENCES.md` as you go.
